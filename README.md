***Filmsmedjan***

Välkommen till vår biosida! Här kan du boka biljetter till väldigt omtyckta klassiker från så tidigt som 1910-talet,
fram till slutet av 1990-talet.

**Installation**

*Klona vårt projekt från [Github] (https://github.com/Martensven/filmvisarna)
*Inne i projektets mapp, skriv in `cd backend` och skriv sedan in `npm install` för att installera alla dependencies. 
*När det är installerat kan du skriva in `npm run devStart` för att backendservern ska starta upp.
*Öppna upp en ny terminal för projektet (detta för att backendservern ska fortsätta köra), i grundmappen skriv även här in `npm install` för att installera dependencies.
*När installationen är klar, skriv in `npm run dev`
*Terminalen ska därefter ge dig länken till localhost-servern och du kan surfa in på hemsidan.
